# ds-list-quiz-notes

## Quiz Questions

Answer the following questions in the provided markdown file before turning in this exercise:

- Briefly describe the List data structure.
  The List data structure holds data in one block, in such a way that all elements have the same size and are accessible by index.
- What are some examples of when you would use a List?
  When you need to access the data quickly.
- How do you read an element of a List given its index? What is its time complexity?
  You read a list element by calling its index. It has a O(1) complexity at most.
- How do you add an element into a List at a given index? What is its time complexity?
  Adding an element to a list requires the enqueue method and has an O(n) complexity.
- How do you update an element in a List at a given index? What is its time complexity?
  You can update the value of an element by removing the outdated entry and then adding the new one. That approach has a O(n) time complexity.
- How do you remove an element from a List at a given index? What is its time complexity?
  You remove an element by using the dequeue property.
